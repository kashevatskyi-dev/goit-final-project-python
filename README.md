# 🤖 Keeply (Персональний помічник)

Keeply — це потужний консольний застосунок для ефективного управління контактами та нотатками. Проєкт розроблено як фінальний командний проєкт з дисципліни "Python Programming", що демонструє використання принципів ООП, чистого коду (PEP 8) та серіалізації даних.

## Головна ідея
У сучасному світі інформація — це найцінніший ресурс. Мета нашого проєкту — створити швидкий, безвідмовний та інтуїтивно зрозумілий інструмент, який працює локально і допомагає користувачу тримати всі важливі контакти та ідеї (нотатки) під рукою. Завдяки розумному аналізу команд, помічник прощає одруківки та робить взаємодію максимально природною.

## Основні можливості (Features)

### 👥 Управління контактами (AddressBook)
- Створення, редагування та видалення контактів.
- Зберігання номерів телефонів (з валідацією формату), електронної пошти (з валідацією Regex) та фізичної адреси.
- Відстеження днів народження: можливість дізнатися, кого треба привітати у найближчі `N` днів.
- Глобальний пошук по контактах (за іменем, телефоном, поштою чи адресою).

### 📓 Управління нотатками (NoteBook)
- Створення, редагування та видалення текстових нотаток.
- Підтримка тегів: додавайте теги прямо в текст нотатки (наприклад, `#робота`).
- Пошук та сортування нотаток за ключовими словами (тегами).
- Повнотекстовий пошук за вмістом нотаток.

### 🧠 Інтелектуальний аналіз та Збереження
- Smart-вгадування команд: Якщо ви зробили одруківку (наприклад, `ad-note` замість `add-note`), помічник зрозуміє це і запропонує правильний варіант.
- Надійність: Жорстка обробка помилок вводу. Програма не "падає" при некоректних даних.
- Збереження стану: Всі дані автоматично зберігаються на жорсткий диск при виході та відновлюються при наступному запуску.

## 👥 Команди для роботи з контактами

- **`add [ім'я] [телефон]`** — додає новий контакт або новий телефон до існуючого.

- **`change [ім'я] [старий] [новий]`** — замінює старий номер телефону на новий.

- **`phone [ім'я]`** — виводить усі телефони контакту.

- **`all <сторінка>`** — показує список контактів на вказаній сторінці.  
  За замовчуванням — перша сторінка.

- **`add-email [ім'я] [email]`** — додає email з перевіркою формату.

- **`add-address [ім'я] [адреса]`** — додає фізичну адресу до контакту.

- **`add-birthday [ім'я] [DD.MM.YYYY]`** — додає дату народження.

- **`show-birthday [ім'я]`** — показує дату народження контакту.

- **`birthdays [кількість_днів]`** — показує контакти з днями народження у найближчі дні (за замовчуванням 7).

- **`search-contact [запит]`** — шукає контакти по всіх полях.

- **`delete-contact [ім'я]`** — видаляє контакт з бази.


## 📝 Команди для роботи з нотатками

- **`add-note [заголовок] [текст] [#тег]`** — створює нову нотатку. Теги додаються через `#`.

- **`all-notes <сторінка>`** — показує список нотаток на вказаній сторінці.  
  За замовчуванням — перша сторінка.

- **`search-note [запит]`** — шукає нотатки у заголовках і тексті.

- **`search-tag [тег]`** — показує нотатки з вказаним тегом.

- **`sort-tags`** — виводить нотатки, відсортовані за тегами.

- **`edit-note [заголовок] [новий_текст]`** — змінює текст існуючої нотатки.

- **`delete-note [заголовок]`** — видаляє нотатку.

- **`add-tag [заголовок] [#тег]`** — додає новий тег до нотатки.

- **`remove-tag [заголовок] [#тег]`** — видаляє тег з нотатки.
