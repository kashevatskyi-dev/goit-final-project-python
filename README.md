# 🤖 Keeply (Персональний помічник)

Keeply — це потужний консольний застосунок для ефективного управління контактами та нотатками. Проєкт розроблено як фінальний командний проєкт з дисципліни "Python Programming", що демонструє використання принципів ООП, чистого коду (PEP 8) та серіалізації даних.

## Головна ідея
У сучасному світі інформація — це найцінніший ресурс. Мета нашого проєкту — створити швидкий, безвідмовний та інтуїтивно зрозумілий інструмент, який працює локально і допомагає користувачу тримати всі важливі контакти та ідеї (нотатки) під рукою. Завдяки розумному аналізу команд, помічник прощає одруківки та робить взаємодію максимально природною.

## Основні можливості (Features)

### 👥 Управління контактами (AddressBook)
- Створення, редагування та видалення контактів.
- Зберігання номерів телефонів (з валідацією формату), електронної пошти (з валідацією Regex) та фізичної адреси.
- Відстеження днів народження: можливість дізнатися, кого треба привітати у найближчі `N` днів.
- Глобальний пошук по контактах (за іменем, телефоном, поштою чи адресою).

### 📓 Управління нотатками (NoteBook)
- Створення, редагування та видалення текстових нотаток.
- Підтримка тегів: додавайте теги прямо в текст нотатки (наприклад, `#робота`).
- Пошук та сортування нотаток за ключовими словами (тегами).
- Повнотекстовий пошук за вмістом нотаток.

### 🧠 Інтелектуальний аналіз та Збереження
- Smart-вгадування команд: Якщо ви зробили одруківку (наприклад, `ad-note` замість `add-note`), помічник зрозуміє це і запропонує правильний варіант.
- Надійність: Жорстка обробка помилок вводу. Програма не "падає" при некоректних даних.
- Збереження стану: Всі дані автоматично зберігаються на жорсткий диск при виході та відновлюються при наступному запуску.

📖 Довідник команд

Системні команди

			1. hello - Отримати привітання від бота.
			2. close або exit - Зберегти всі дані та закрити програму.

Команди для роботи з контактами

			1. Додати контакт: 
						формат: add [ім'я] [телефон], 
						опис: Додає новий контакт або новий телефон до існуючого.

			2. Змінити телефон:
						формат: change [ім'я] [старий] [новий],
						опис: Замінює старий номер на новий.

			3. Показати телефон:
						формат: phone [ім'я],
						опис: Виводить усі телефони контакту.

			4. Всі контакти:
						формат: all <сторінка>,
						опис: Виводить список усіх контактів на вказаній сторінці. 
                              За замовчуванням виводить першу  сторінку.

			5. Додати Email:
						формат: add-email [ім'я] [email],
						опис: Додає email (з перевіркою формату).

			6. Додати адресу:
						формат: add-address [ім'я] [адреса],
						опис: Додає фізичну адресу.

			7. Додати ДН:
						формат: add-birthday [ім'я] [DD.MM.YYYY],
						опис: Додає дату народження.

			8. Показати ДН:
						формат: show-birthday [ім'я],
						опис: Показує дату народження контакту.

			9. Дні народження:
						формат: birthdays [кількість_днів],
						опис: Показує, у кого ДН в найближчі дні (за замовчуванням 7).

			10. Пошук контакту:
						формат: search-contact [запит],
						опис: Шукає збіги по всіх полях контактів.

			11. Видалити:
						формат: delete-contact [ім'я],
						опис: Видаляє контакт з бази.

Команди для роботи з нотатками

			1. Додати нотатку:
						формат: add-note [заголовок] [текст] [#тег],
						опис: Створює нотатку. Теги додаються через #.

			2. Всі нотатки:
						формат: all-notes <сторінка>,
						опис: Виводить список усіх нотаток на вказаній сторінці. 
                              За замовчуванням виводить першу  сторінку.

			3. Пошук у тексті:
						формат: search-note [запит],
						опис: Пошук по заголовках і вмісту нотаток.

			4. Пошук за тегом:
						формат: search-tag [тег],
						опис: Виводить нотатки, які містять вказаний тег.

			5. Сортування:
						формат: sort-tags,
						опис: Виводить нотатки, відсортовані за тегами.

			6. Редагувати:
						формат: edit-note [заголовок] [новий_текст],
						опис: Повністю змінює текст існуючої нотатки.

			7. Видалити: 
						формат: delete-note [заголовок],
						опис: Видаляє нотатку з бази.

			8. Додати тег: 
						формат: add-tag [заголовок] [#тег],
						опис: Створює новий тег для нотатки у базі.

			9. Видалити тег:
						формат: remove-tag [заголовок] [#тег],
						опис: Видаляє тег для нотатки з бази.
